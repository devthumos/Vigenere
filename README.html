Vigenere Devthumos Tool

Descrição

Esta ferramenta foi criada para cifrar e decifrar mensagens usando a cifra de Vigenère. Além disso, oferece funcionalidades para encontrar tamanhos de chave prováveis e analisar visualmente as distribuições de grupos de acordo com as ocorrências de trigramas no Power BI.

Requisitos

  • Python 3.x
  • Pandas
  • Power BI (para visualização das distribuições)

Utilização

  1. Clone ou baixe este repositório para o seu computador.

  2. Instale as bibliotecas necessárias executando o seguinte comando no terminal ou prompt de comando:

bash
pip install pandas
  1. Execute o script main.py no terminal ou prompt de comando:
bash
python main.py
  1. Siga as instruções exibidas na tela para selecionar a ação desejada:
markdown
______ Vigenere Devthumos Tool ______
    1) Encriptar
    2) Decriptar
    3) Probabilidades de Tamanhos de Chave
    4) Distribuicao dos Agrupamentos em Power BI
    5) Sair

Encriptar

  1. Escolha a opção 1.
  2. Insira a mensagem a ser cifrada no arquivo Mensagens/Mensagem_Original.txt.
  3. Digite a chave desejada.
  4. A mensagem cifrada será gerada no arquivo Mensagens/Mensagem_Cifrada.txt.

Decriptar

  1. Escolha a opção 2.
  2. Insira a mensagem cifrada no arquivo Mensagens/Mensagem_Cifrada.txt.
  3. Digite a chave desejada.
  4. A mensagem decifrada será gerada no arquivo Mensagens/Mensagem_Decifrada.txt.

Probabilidades de Tamanhos de Chave

  1. Escolha a opção 3.
  2. Insira a mensagem cifrada no arquivo Mensagens/Mensagem_Cifrada.txt.
  3. Digite o número mínimo de ocorrências de trigramas desejado.
  4. As probabilidades de tamanhos de chave serão geradas no arquivo KeySize/SizeScore.txt.

Distribuição dos Agrupamentos em Power BI

  1. Escolha a opção 4.
  2. Insira a mensagem cifrada no arquivo Mensagens/Mensagem_Cifrada.txt.
  3. Digite o tamanho provável da chave.
  4. A distribuição dos agrupamentos será gerada no arquivo Distributions/distribution.csv.
  5. Abra o arquivo Dashboards/dashboard_de_distribuicao.pbix no Power BI para visualizar as distribuições.

Nota

As pastas e arquivos relevantes para cada função são:

  • Mensagens: Contém os arquivos de texto para as mensagens originais, cifradas e decifradas.
  • KeySize: Contém o arquivo de texto com as probabilidades de tamanhos de chave.
  • Distributions: Contém o arquivo CSV com as distribuições de agrupamentos.
  • Dashboards: Contém o arquivo do Power BI para visualizar as distribuições.
